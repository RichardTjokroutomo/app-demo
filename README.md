# Daily News - CNA Style Mobile App

A mobile-optimized news recommendation page styled after Channel News Asia (CNA), designed for iPhone 14 form factor (390x844px).

## Features

- Responsive design for iPhone 14 dimensions
- CNA-inspired color scheme (dark blue with red accents)
- Breaking news banner
- Featured story with overlay
- Five news recommendation cards with categories
- Interactive elements (clickable cards, menu/search buttons)
- Mobile-friendly navigation and footer

## Files

- `index.html` - Main HTML structure
- `css/style.css` - All styling
- `README.md` - This file

## Styling Details

- **Colors**:
  - Primary: `#0a2463` (CNA dark blue)
  - Accent: `#e63946` (CNA red)
  - Background: `#f8f9fa`
  - Text: `#333`

- **Typography**:
  - Headers: Playfair Display (serif)
  - Body: Inter (sans-serif)

- **Layout**:
  - Max-width: 390px (iPhone 14 width)
  - Min-height: 844px (iPhone 14 height)
  - Centered with shadow to simulate phone screen

## How to View

Simply open `index.html` in any modern web browser. The page is fully self-contained and uses:

- Font Awesome icons (CDN)
- Google Fonts (Inter, Playfair Display)
- Unsplash images (CDN)

## Interactive Elements

- Click any news card to trigger an alert
- Menu and search buttons show alerts
- Hover effects on cards (desktop)
- Active state on navigation

## Browser Support

Works on all modern browsers with support for CSS Flexbox and Grid.
